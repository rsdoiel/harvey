harvey

Harvey is an agent similar to Claude Code but written in Go and designed to use Ollama server to access large models locally or if configurated access publicai.co. It is a terminal based application.

The Harvey name is inspired by the Mary Chase play of the same name. My little agent runs on small computers but is available to those who choose to see its value. Harvey in the play was a Púca, a mythic Celtic spirit who at times was proned to mischief. The Púca chose who could see it but even then the person doing the seeing had to be willing to see it too.

I think the current AI hype cycle will likely end with a bang and a crash. The open models will still be here, we’ll just need to be able to run them. The problem is that really big models require big energy expensive computers. I think smaller models, ones that can run at the edge will be the ones that carry weight in the long wrong. To explore the idea of small models on small computers I dreamed up Harvey. We’ll have to wait to see where that adventure could leads.
