About this software

harvey 0.0.0

concept

Authors

  • R. S. Doiel, https://orcid.org/0000-0003-0900-6903

Harvey is an agent similar to Claude Code but written in Go and designed to use Ollama server to access large language models locally or if configurated access publicai.co and the Abertus model. It is a terminal based application.

  • GitHub: https://github.com/rsdoiel/harvey
  • Issues: https://github.com/rsdoiel/harvey/issues
