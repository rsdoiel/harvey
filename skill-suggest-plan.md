### W1 — Implement `renderSkillMD` + `latestSessionFile`

**Goal:** Pure functions only — no LLM call. These are deterministic.

`renderSkillMD(c SkillCandidate, author string) string`:
- Uses `strings.Builder` to render the SKILL.md template from design doc
- Variables block: only emitted when `len(c.Variables) > 0`
- Steps block: numbered list from `c.Steps`
- Appends the "Generated by `/skill suggest`" disclaimer

`latestSessionFile(sessionsDir string) (string, error)`:
- `os.ReadDir` on `sessionsDir`
- Filter to entries ending in `.spmd`
- Pick the one with the highest `ModTime()`
- Return full path or error if none found

Acceptance: `go test -run TestRenderSkillMD TestSuggestor_LatestSessionFile ./...` passes.

### W2 — Implement `Suggest` (LLM extraction + interactive review)

**Goal:** Full `Suggest` pipeline.

`Suggest(ctx context.Context, sessionPath string, agent *Agent, out io.Writer, in io.Reader) error`:

1. If `sessionPath == ""`, call `latestSessionFile(ws.agentsDir() + "/sessions")`.
2. `os.ReadFile(sessionPath)` → session text.
3. Call `agent.Client.Chat(ctx, []Message{{system, skillSuggestorPrompt}, {user, sessionText}}, &buf)`.
4. Extract JSON from `buf.String()` using `extractJSON` (already in `memory_miner.go`; make it package-level if needed).
5. `json.Unmarshal` → `[]SkillCandidate`.
6. If empty, print "No skill candidates found in this session." and return.
7. For each candidate, print a summary and prompt `Accept? [y/n/q]`:
   - `y` → call `writeSkillMD(ws, c)` then print `"  Written: agents/skills/<name>/SKILL.md"`
   - `n` → print `"  Skipped."` and continue
   - `q` or empty → return immediately

`writeSkillMD(ws *Workspace, c SkillCandidate) error`:
- Determines author from `git config user.name` (shell out; fallback "unknown")
- `ws.MkdirAll("agents/skills/" + c.Name + "/scripts")`
- `ws.WriteFile("agents/skills/" + c.Name + "/SKILL.md", []byte(renderSkillMD(c, author)), 0644)`

Acceptance: running `/skill suggest` on a real session file produces one or more SKILL.md stubs.

## Risks

| Risk | Mitigation |
|---|---|
| LLM returns malformed JSON | `extractJSON` + `json.Unmarshal` already handle this; return "no candidates" on parse failure |
| Skill name collides with existing skill | Check `ws.AgentsDir()+"/skills/"+c.Name` before writing; prompt user to rename or skip |
| Session file is too large for LLM context | Use `RunChunkedAnalysis` or truncate to first N bytes with a note; defer for now |
| `a.Client` is nil (no backend) | Guard: if `agent.Client == nil`, return `fmt.Errorf("no LLM backend connected")` |
