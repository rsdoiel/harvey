I would like the following behavior to be available in Harvey's REPL. Some of these will need to be implemented in termlib module others should be implemented in the Harvey project. Please review the list and identify where is feature fits and make a plan for updating termlib and harvey accordingly.


- Multi-line Input: Press Ctrl+J or Shift+Enter for select terminals to insert a newline.
- File Paths: Reference files in your prompt using the @ symbol for smart autocompletion (e.g., > Read the file @src/agent.py).
- External Editor: Press Ctrl+G to edit your current input in an external editor.
