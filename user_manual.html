Harvey User Manual

Harvey is a terminal-based coding agent backed by a local Ollama server or publicai.co. It provides an interactive REPL where you chat with a large language model while also being able to read files, run commands, search code, and apply suggested changes — all sandboxed to a single workspace directory.
