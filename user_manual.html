Harvey User Manual

Harvey is a terminal-based coding agent backed by a local Ollama server (or any provider supported by mozilla-ai/any-llm-go). It provides an interactive REPL where you chat with a large language model while also reading files, running commands, searching code, and applying suggested changes, all sandboxed to a single workspace directory.
