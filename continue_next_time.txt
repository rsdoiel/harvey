HOW TO PICK THIS UP
-------------------
Whichever direction is chosen needs its own design -> decisions -> planning
cycle before implementation, same as every item in this session: a
<topic>-design.md, explicit decisions confirmed before coding (not assumed),
a <topic>-plan.md, then TDD-first implementation with each decision logged
in DECISIONS.md. Don't skip the plan artifact even for small-seeming work —
that was a real mistake caught mid-session this time.
